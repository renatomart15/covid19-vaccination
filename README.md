Projeto desenvolvido com React e Tailwind CSS para praticar a metodologia Mobile First, onde fiz a estilização base para telas pequenas, adicionando responsividade conforme a tela cresce. Para trazer uma experiência fluida, utilizei o framer-motion para pequenas animações, o react-icons para os ícones e elementos da página e o react-scroll para uma navegação suave entre as seções.

### 🛠️ Como rodar

```
git clone https://github.com/seu-usuario/seu-repo.git
npm install
npm run dev
```
